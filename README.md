# Islamic Quote Generator

## 📌 Description
Islamic Quote Generator is a simple Python-based project that displays
random Islamic quotes using lists. It is designed to demonstrate basic
Python concepts such as lists, loops, and random selection.

## 🚀 Features
- Displays random Islamic quotes
- Simple and clean console output
- Beginner-friendly Python project
- Easy to modify and extend

## 🛠 Technologies Used
- Python 3
- Git
- GitHub

## ▶ How to Run the Project

1. Make sure Python is installed.
2. Open terminal inside project folder.
3. Run:

python main.py

(Replace main.py with your actual file name if different.)
